# [portfolio](https://kaizhang.ca/)

---

Simple project showcase and parody of the Spotify artist page.

Viewing projects for an extended duration "streams them", with an up-do-date popular projects list at the top.

Built with _React_ and _TailwindCSS_, streams saved in _Firestore_, and deployed to _GH-pages_ with a simple _CD pipeline_.
